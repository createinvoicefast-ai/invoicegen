Privacy Policy

How this site handles data.

This project is built as a front-end experience. Account records, invoices, and draft data are stored locally in the browser so the user can access personal billing information without a server.

Data we store

  • Name, email, and password hash for the local login system.
  • Invoice drafts, saved invoices, and client details tied to the signed-in account.
  • Theme preference and basic UI settings for a better experience.

How data is used

Stored data is used only to power login, dashboard history, invoice editing, and template preferences on this device.

Ads and analytics

The layout includes reserved space for future advertising but does not show ads now. If analytics or ad services are added later, this policy should be updated to describe the third-party tools being used.
